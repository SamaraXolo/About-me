Acerca de mí

        Proyecto web estático desarrollado con HTML5 y CSS3 como parte de mi formación en desarrollo frontend.

Descripción

        “Acerca de mí” es una página web personal que presenta información básica mediante una estructura semántica clara y un diseño visual limpio. El proyecto incluye:

                Encabezado con imagen de fondo y overlay

                Sección de tarjetas con información personal

                Diseño responsive utilizando Flexbox

                Organización estructurada de archivos (HTML, CSS, assets)

Tecnologías utilizadas

                HTML5 (estructura semántica)

                CSS3 (Flexbox, box-sizing, background properties)

                Fuentes personalizadas con @font-face

                Organización de assets (images, fonts)

Objetivo

        Practicar maquetación, estructura semántica, posicionamiento con Flexbox y aplicación de estilos consistentes siguiendo un brief técnico.
